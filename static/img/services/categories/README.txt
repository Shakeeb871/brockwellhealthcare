Homepage "Our Services" category images
=======================================

Drop one .webp file per service category here, named EXACTLY by the category
slug (all lowercase, hyphens, no spaces). The homepage slider picks them up
automatically — no admin or code changes needed.

Your images  ->  save them here as:

  Regenerative Wellness.webp   ->  regenerative-wellness.webp
  Regenerative Medicine.webp   ->  regenerative-medicine.webp
  Longevity Healthspan.webp    ->  longevity-healthspan.webp
  Anti-Aging Aesthetics.webp   ->  anti-aging-aesthetics.webp
  Advanced Diagnostics.webp    ->  advanced-diagnostics.webp
  Emsculpt For Pain.webp       ->  emsculpt-for-pain.webp

Recommended: 16:10 ratio (e.g. 800x500), optimised .webp.

After adding the files: commit + push, then run deploy.sh (which runs
collectstatic). If a file is missing, that card simply falls back to the
category's admin Image, then its icon — nothing breaks.
